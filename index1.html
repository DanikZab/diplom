Мой путь в веб-разработке и 3D-моделировании

Добро пожаловать на мой сайт! Здесь я расскажу о своем опыте обучения и создания проектов.

Обо мне

Меня зовут Даниил, и я увлечен веб-разработкой и 3D-моделированием. Мой путь начался несколько лет назад, когда я впервые открыл для себя мир программирования. С тех пор я постоянно учусь и совершенствую свои навыки.

Веб-разработка привлекла меня своей универсальностью. С помощью HTML, CSS и JavaScript можно создавать что угодно: от простых статических страниц до сложных веб-приложений. Я начал с основ, изучая теги HTML и свойства CSS, а затем перешел к более сложным темам, таким как анимации, работа с API и создание интерактивных элементов.

Одним из самых интересных моментов было изучение CSS-анимаций. Например, я создал кнопку, которая меняет цвет и увеличивается при наведении: 


.button {
    background-color: #3498db;
    color: white;
    padding: 10px 20px;
    border: none;
    transition: background-color 0.3s, transform 0.3s;
}

.button:hover {
    background-color: #2980b9;
    transform: scale(1.1);
}

Также я изучал JavaScript и создавал динамические элементы. Например, вот простой код для отображения текущего времени на странице: 


function showTime() {
    const now = new Date();
    const timeElement = document.getElementById('time');
    timeElement.textContent = now.toLocaleTimeString();
}

setInterval(showTime, 1000);

Мои проекты

Помимо веб-разработки, я увлекаюсь созданием 3D-моделей. Мне нравится работать в программах, таких как Blender, и создавать реалистичные объекты. Например, я создал 3D-модель дома с детализированным интерьером. Вот пример кода для создания простого куба в Blender: 


import bpy

# Создание куба
bpy.ops.mesh.primitive_cube_add(size=2, location=(0, 0, 0))

Также я изучал текстурирование и освещение, чтобы сделать модели более реалистичными. Вот пример настройки освещения: 


import bpy

# Добавление источника света
bpy.ops.object.light_add(type='SUN', location=(10, 10, 10))

Я также создал несколько проектов, где объединил веб-разработку и 3D-моделирование. Например, я сделал сайт, на котором можно вращать 3D-модель прямо в браузере с помощью библиотеки Three.js: 


const scene = new THREE.Scene();
const camera = new THREE.PerspectiveCamera(75, window.innerWidth / window.innerHeight, 0.1, 1000);

const renderer = new THREE.WebGLRenderer();
renderer.setSize(window.innerWidth, window.innerHeight);
document.body.appendChild(renderer.domElement);

const geometry = new THREE.BoxGeometry();
const material = new THREE.MeshBasicMaterial({ color: 0x00ff00 });
const cube = new THREE.Mesh(geometry, material);
scene.add(cube);

camera.position.z = 5;

function animate() {
    requestAnimationFrame(animate);
    cube.rotation.x += 0.01;
    cube.rotation.y += 0.01;
    renderer.render(scene, camera);
}

animate();
