Вот такую вот таблицу календарь вы сможете сделать с помощью html и css

Для этого нужно понять, как вообще строится таблица

Шаг 1 Добавление заголовка

<table>tr><th colspan="5">Мои рабочие дни, Май 2024</th></tr></table>

Шаг 2 Добавление столбиков

<table>tr><th colspan="5">Мои рабочие дни, Май 2024</th></tr>< <tr> <td>Понедельник</td> <td>Вторник</td> <td>Среда</td> <td>Четверг</td> <td>Пятница</td> </tr> /table>

Шаг 3 Добавление строк

<table>tr><th colspan="5">Мои рабочие дни, Май 2024</th></tr>< <tr> <td>Понедельник</td> <td>Вторник</td> <td>Среда</td> <td>Четверг</td> <td>Пятница</td> </tr> <tr> <td>2 </td> <td>3 </td> <td>4 </td> <td>5 </td> <td>6 </td> </tr> <tr> <td>9 </td> <td>10 </td> <td>11 </td> <td>12 </td> <td>13 </td> </tr> <tr> <td>16 </td> <td>17 </td> <td>18 </td> <td>19 </td> <td>20 </td> </tr> <tr> <td>23 </td> <td>24 </td> <td>25 </td> <td>26 </td> <td>27 </td> </tr> <tr> <td>30 </td> <td>31 </td> <td colspan="3"> </td> </tr> /table>

Шаг 4 Добавление оформления CSS(черный промежуток это новый абзац)

table { font-family: "Lucida Sans Unicode", "Lucida Grande", Sans-Serif; text-align: center; border-collapse: collapse; border-spacing: 5px; background: #E1E3E0; border-radius: 20px; } th { font-size: 22px; font-weight: 300; padding: 12px 10px; border-bottom: 2px solid #F56433; color: #F56433; } tbody tr:nth-child(2) { border-bottom: 2px solid #F56433; } td { padding: 10px; color: #8D8173; }

После всего этого кода у вас должна получиться такая же таблица.
